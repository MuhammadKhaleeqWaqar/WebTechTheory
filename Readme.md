# 📌 Project Description
SkillsBar is a simple, modern, and responsive frontend promotional website built for a startup learning platform. The project is developed using HTML5 and CSS3 only, without using JavaScript or backend technologies. It showcases course offerings, company information, registration form UI, and contact page design.

The goal of this project is to demonstrate clean UI design, responsive layout techniques, and modern startup-style website structure suitable for student portfolio or academic submission.

---

## ✨ Features
- Fully responsive design
- CSS-only dropdown navigation
- CSS animation carousel
- Modal popup using :target selector
- Course listing cards with hover effects
- Student registration form UI with validation styling
- About section with team images
- Google Map iframe integration
- Gradient hero section
- Clean startup-style layout
